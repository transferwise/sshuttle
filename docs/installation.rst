Installation
============

- From PyPI::

      pip install tshuttle

- Debain package manager::

      sudo apt install tshuttle

- Clone::

      git clone https://github.com/tshuttle/tshuttle.git
      cd tshuttle
      ./setup.py install


Optionally after installation
-----------------------------

- Add to sudoers file::

      tshuttle --sudoers
